## Quick Reference

### Brand Hierarchy

- **Stria** = Brand (parent company/studio)
- **Drift** = First product (Max for Live phase sequencer)
- **Future products** = Single-word names (Fold, Shift, Loop, etc.)

### Core Positioning

**"Logic, not AI"** — Artist-driven algorithmic tools with transparent, deterministic rules. Research-grade, DMA-led, human-designed.

### Voice Attributes

- **Clear**: Say what it is; no vague "generative magic"
- **Confident**: Research-grade; no hype or oversell
- **Warm**: Invite curiosity and play
- **Expert**: Reference process music, algorithms, craft

### Product Naming Pattern

Single evocative words, not descriptive. Examples: Drift, Fold, Shift, Loop.

## Usage

These docs serve as the single source of truth for brand decisions across all Stria products. When creating new products, marketing materials, or user-facing content, consult these guidelines first.
